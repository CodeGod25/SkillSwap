# Images Folder

## Required Image

**`hero-community.jpg`** - Landing page hero image

### Image Details:
- Shows happy community members socializing outdoors
- Should be a warm, inviting photo of people connecting and helping each other
- Recommended dimensions: 1200x1200px (square aspect ratio)
- Format: JPG or PNG
- Optimized for web (under 500KB recommended)

### How to Add:
1. Save your community image as `hero-community.jpg`
2. Place it in this folder: `frontend/public/images/`
3. The Next.js Image component will automatically optimize it

### Current Usage:
- Used in: `components/stitch/HeroMarketing.jsx`
- Alt text: "Happy community members socializing and trading skills"
- Display: Fills the hero section on the landing page with rounded corners and border
